Object-Oriented Programming(OOP) in JavaScript

It simply means separating the code into objects that have properties and methods. The objects can be reused or easily modified, as required. The three pillars of OOP's are: Encapsulation, Polymorphism, and Inheritance.

Encapsulation: The inner workings that are kept hidden inside the object and only the essential functionalities are exposed to the user. For example, in our IDE we print to the console and it prints the result to us, we don't know what is happening behind the hood.

Polymorphism: The same process that can be used for different objects. Objects can share the same method, but also have the ability to override shared methods with a more specific implementation.

Inheritance: We can taje an object that already exists and inherit all its properties and methods. Then we can improve on its functionality by adding new properties and methods

JavaScript contains a number of built-in constructor such as Object, Array and Function

Prototypal Inheritance

- JavaScript uses a prototypal inheritance model which means that every class has a prototype property that is shared by every instance of the class. So any properties or methods of a class prototype can be accessed by every object instantiated by that class.

Forms

Forms are made up of a element that contains form controls such as input fields, select menus and buttons. These fields can be populated with the information that is processed once the form has been submitted.
Document.Forms returns an HTML collectin of all the forms in the document in the ordery they appear in the markup.

const form = document.forms[0];

A form object also has a method called elements that returns an HTML collection of all the elements contained in the form.
const [input,button] = form.elements;

The most important form event is the submit event, occurring when the form is submitted. Usually, this will send the content of the form to the server to be processed.

Form Validation: A process of checking whether a user has entered the information into a form correctly. Example: A required field, an email address is valid, a number is entered, password is at least a minimum number of characters.
